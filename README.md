# 系统监测器
## 技术栈

- rust
- tauri
- vue3
- vue-echarts
- rsbuild

## 软件介绍

用于查看电脑的硬件信息，实时监测cpu，内存，硬盘，网络，进程等系统资源

图形化，动态化展示，美观实用
